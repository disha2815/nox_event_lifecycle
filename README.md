# Event Lifecycle and Certification

## Overview
Nox Event Lifecycle is a structured event management system designed to handle the complete lifecycle of events—from creation and scheduling to execution and closure. The project focuses on building a scalable and organized workflow for managing events efficiently.

This project demonstrates practical implementation of full-stack development concepts, system design thinking, and clean architecture principles.

---

## Features
- Create and manage events
- Track event lifecycle status (Created → Scheduled → Ongoing → Completed)
- Organized and modular project structure
- Easy scalability for future enhancements
- Clean and readable codebase

## Getting Started

### Step 1: Clone the repository
```bash
git clone https://github.com/disha2815/nox_event_lifecycle.git

### Step 2: Go to project folder
cd nox_event_lifecycle

### Step 3: Install dependencies

For Node.js:
npm install

For Python:
pip install -r requirements.txt

---

### Step 4: Run the project

For Node.js:
npm start

For Django:
python manage.py runserver
